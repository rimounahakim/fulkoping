# Fulköpings bibliotek

## Beskrivning

Detta är en enkel webbapplikation som är byggd i Java med Spring Boot.
Projektet använder MVC-mönstret (Model, View, Controller).

Applikationen gör det möjligt för användare att söka efter böcker,
logga in och låna böcker via webbsidan.

Projektet är gjort som en inlämningsuppgift i kursen Webbutveckling i Java.

---

## Funktioner

- Söka efter böcker
- Se bokdetaljer
- Logga in med användarnamn och lösenord
- Låna bok
- Lämna tillbaka bok
- Se mina aktiva lån (Mina lån)
- Se lånehistorik (Historik)

---

## Hur man startar projektet lokalt

1. Klona projektet från GitHub
2. Kör i terminalen:

   ./mvnw clean package

3. Starta applikationen:

   ./mvnw spring-boot:run

4. Öppna webbläsaren och gå till:

   http://localhost:8081
